CDI

A Injeção de Dependência e Contextos (CDI), especificada por JSR-299, é parte integrante do Java EE 6

e fornece uma arquitetura que permite aos componentes do Java EE, como os servlets, enterprise beans e JavaBeans,

existirem dentro do ciclo de vida de uma aplicação com escopos bem definidos.

Além disso, os serviços CDI permitem que os componentes do Java EE, como beans de sessão EJB e beans gerenciados do JavaServer Faces (JSF),

sejam injetados e interajam de maneira acoplada flexível, disparando e observando eventos
